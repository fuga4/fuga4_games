# ココロほどくゲーム集 (fuga4_games)

忙しい子育て世代の親御さんが、ちょっとした合間に頭を空っぽにして、心をゆるめるためのスマホ向け癒やしミニゲーム集です。
GitHub Pagesへの自動デプロイワークフローがあらかじめ設定されており、新しいゲームを随時追加していける拡張性の高い設計になっています。

## 🌟 特徴
- **癒やしのパステルダークデザイン**: 寝かしつけ後の暗い部屋でも目に優しい、穏やかな夜空や深海をイメージしたカラーパレット。
- **直感的で気持ちいい操作**: `framer-motion` による滑らかなアニメーションと、スマホでのタップ・ドラッグに最適化されたUI。
- **時間制限・ペナルティなし**: ゲームオーバーはなく、ただ心地よい作業やシミュレーションを自分のペースで楽しめます。

## 🎮 収録ミニゲーム
1. **ほっこりアクアリウム**
   - 画面をタップして「しずく」を集め、きいろい魚やクラゲ、ウミガメなどの生き物を深海に迎え入れるタップ＆放置ゲーム。
   - ゆったりと泳ぐ魚たちをタップすると、可愛いアクションを楽しめます。
2. **すっきりお片付け**
   - 散らばったブロック（●、■、▲、★）をドラッグして、同じ形のカゴに片付ける整理整頓ゲーム。
   - ピタッとはまる感覚と、すべてのブロックが綺麗に片付くことによる「スッキリ感」を体験できます。

## 🛠️ 技術スタック
- **フレームワーク**: React + TypeScript + Vite
- **アニメーション**: Framer Motion
- **アイコン**: Lucide React
- **デプロイ**: GitHub Actions (GitHub Pagesへ自動デプロイ)

## 🚀 開発と実行

### 依存関係のインストール
```bash
npm install
```

### 開発用サーバーの起動 (Local)
```bash
npm run dev
```

### 本番用ビルドと型チェック
```bash
npm run build
```

## 📦 新しいゲームを追加する方法
新しいゲームを開発してこのポータルに追加する手順は非常にシンプルです：

1. **ゲームコンポーネントの作成**:
   - `src/components/games/` ディレクトリ配下に新しいフォルダを作成し、ゲームのコンポーネント（例: `MyNewGame.tsx`）を実装します。
   - `onBackToPortal: () => void` を受け取るPropsを定義し、ホーム画面に戻れるようにします。

2. **メタデータの追加**:
   - `src/data/games.ts` の `gamesList` 配列に新しいゲームの情報を追加します。
     ```typescript
     {
       id: 'my-new-game',
       title: '新しいゲーム',
       description: 'ゲームの簡単な説明。',
       icon: 'Sparkles', // Lucide iconの文字列表現
       accentColor: 'var(--color-peach)',
       tags: ['タグ1', 'タグ2'],
       playTime: '1分〜'
     }
     ```

3. **GameContainer への登録**:
   - `src/components/GameContainer.tsx` 内の `switch (gameId)` 文に新しいゲームのケースを追加します。
     ```typescript
     case 'my-new-game':
       return <MyNewGame onBackToPortal={onBackToPortal} />;
     ```
